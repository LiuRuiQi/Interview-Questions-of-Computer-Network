 4. 网络分层模型

	ISO/OSI (Open System Interconnection) 的七层模型：从下到上是物理层、数据链路层、网络层、传输层、会话层、表示层、应用层。
	
	TCP/IP的五层模型：物理层，数据链路层，网络层，传输层，应用层。

 5. 各种网络设备都工作在哪一层？

	物理层设备：中继器，集线器，双绞线等传输线

	数据链路层设备：网桥（现已很少使用），以太网交换机（二层交换机），网卡（属于MAC层，介于物理层和数据链路层之间，或理解为数据链路层的子层）

	网路层设备：路由器，三层交换机

	传输层：也有工作在四层的交换机和路由器。
